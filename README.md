# rover-computer-vision
Lane Detection: Canny Edge Detection ve Hough Transform kullanarak otonom sürüş için şerit belirleme.

Object Tracking: Renk filtreleme (HSV color masking) veya Cascade Classifier ile belirli nesneleri takip etme.

Pre-processing: Görüntü iyileştirme için Gaussian Blur, Thresholding ve Morphological Operations (Erosion/Dilation) kullanımı.
